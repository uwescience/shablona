Welcome to toymir's documentation!
====================================

`toymir` is a Python package for use within the ISMIR 2018 Open Source and Reproducibility tutorial session.

It is based on the `Shablona <https://github.com/uwescience/shablona>`_ package developed by Ariel Rokem at the University of Washington's eScience Institute.

This is an example of documentation of the software, using sphinx_.
